Private Eye

Private Eye is an NVDA global plugin that periodically captures the screen and asks an Ollama vision model for short descriptions.

Commands

  • NVDA+Alt+V: Toggle live description.
  • NVDA+Shift+Alt+V: Clear rolling context and restart if active.
  • NVDA+Control+Alt+V: Open Private Eye settings.

Defaults

  • Ollama server: http://localhost:11434
  • Model: gemma4
  • Prompt mode: Automatic, with manual General, Video, and Game modes.
  • General system prompt: configurable in Private Eye settings.
  • Capture interval: 4 seconds
  • Ollama request timeout: 60 seconds
  • Context entries: 8

Notes

The add-on requires an Ollama server with a vision-capable model. It does not bundle or install Ollama models.
